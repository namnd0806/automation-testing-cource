## 💼 CAREER PATH

### Entry Level
**Automation Tester**
- Salary: 15-25 triệu/tháng
- Viết và maintain automation scripts

### Mid Level (2-4 years)
**Senior Automation Engineer**
- Salary: 25-40 triệu/tháng
- Design framework, mentor juniors

### Senior Level (4+ years)
**SDET** (Software Development Engineer in Test)
- Salary: 40-70 triệu/tháng
- Build tools, CI/CD, test architecture
